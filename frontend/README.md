# NG-Boilerplate

### Beinhaltet
* Webpack
* AoT
* Karma und Mocha
* E2E - testcafe
* Hot Module Replacement

### QuickStart
** Node version >=6.0 und NPM >=3 werden benÃ¶tigt**
> Installation

```bash
# clone
git clone git@git.testo-web.de:testo-microservices/ng-boilerplate.git

# install
npm install

# start
npm start

# Hot Modul Replacement
npm run srever:dev:hmr
```

Der Dienst ist zu erreichen unter http://localhost:8081

> Tests

```bash
# unit test
npm run test

# e2e test
npm run e2e
npm run e2e:firfox
```
